## 🧑‍💻 Installation
To install, you must download repository and dependencies:
```
git clone https://github.com/jDan735/kanobu.git
cd kanobu
pip3 install -r requirements.txt
```
## 🚀 Start
Command to start:
```
python3 ropasci.py
```
